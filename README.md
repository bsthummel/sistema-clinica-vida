# Sistema Clinica Vida

  O presente documento consolida os resultados do projeto integrado que visa 
informatizar os processos de atendimento médico da Clínica Vida+. A transição de 
processos manuais para um sistema digital é um passo crucial para otimizar a 
eficiência operacional, melhorar o controle de dados dos pacientes e garantir a 
agilidade no atendimento. 

  O desafio proposto abrange diversas facetas do desenvolvimento de software 
e análise de sistemas, combinando práticas de gestão de projetos ágeis (Scrum), 
lógica de programação em Python, análise de requisitos e modelagem de sistemas 
(UML). 

  O objetivo principal deste projeto é conceber um sistema funcional e bem 
estruturado que permita à equipe da clínica gerenciar o fluxo de pacientes de forma 
eficaz. Especificamente, o sistema deve suportar as atividades da secretária — como 
cadastro, agendamento, confirmação e cancelamento de consultas — e as ações do 
médico, como o cancelamento de consultas e a geração de receitas médicas.
